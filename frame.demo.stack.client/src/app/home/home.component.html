Built, quite easily, on Frame!

This project, which started as a bet between Andrew Morgan and Ruben Spruijt was to demonstrate how easy (or not) it was to leverage frame as a virtual desktop for developer use cases. This demo project consists of a number of typical developer components, all wired together while using a Frame Virtual desktop.

Despite not wanting to lose, here we are and it was quite a pleasurable experience!
